A free plugin that integrates some helpful import/export functionality into Theme Blvd themes.

== Description ==

**NOTE: This plugin requires Theme Blvd framework v2.5.2+**

When using a Theme Blvd theme, this plugin adds some helpful import/export functionality.

== Installation ==

1. Upload `theme-blvd-importer` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
